# Design System & Editorial Guidelines

## Typography

### Inter
Used for:
- navigation
- headings
- UI components
- figure labels
- metadata
- diagrams

Recommended weights:
- 400
- 500
- 600
- 700
- 900

### Source Serif 4
Used for:
- long-form essays
- reflective writing
- quotations
- reading-focused sections

### IBM Plex Mono
Used for:
- code blocks
- annotations
- references
- technical metadata
- coordinates
- axis labels

## Design Principles

The interface should feel:
- architectural
- restrained
- systematic
- and intellectually focused

Avoid:
- dashboard aesthetics
- decorative gradients
- excessive shadows
- startup-style UI patterns
- ornamental animations

Preferred characteristics:
- visible grid systems
- asymmetrical layouts
- generous whitespace
- typographic hierarchy
- restrained motion
- SVG-first graphics
